定义变量

val和var

val/var 变量标识：变量类型 = 初始值

  • val 不可重新赋值；
  • var 可以重新赋值。
val name:String = "Tom"
val grades:Int = 80

类型推断定义变量

val name = "Tom"
val grades = 80

scala可以自动根据变量的值来推断变量类型。

惰性赋值

lazy val/var 变量名 = 表达式
在存储SQL语句时，我们不希望在定义的时候就加载它，而只在用到该变量时才加载，这时就需要惰性赋值。

lazy val sql = 
"""
{SQL语句}
"""

字符串

使用双引号

val/var 变量名 = “字符串”

val name = "hadoop"
println(name + name.length)
----------------------------
>hadoop6

插值表达式

val/var 变量名 = s"${变量/表达式}字符串"

val name = "zhangsan"
val age = 30
val sex = "male"
val info = s"name=${name}, age=${age}, sex=${sex}"
println(info)
----------------------------
name=zhangsan, age=30, sex=male

使用三引号

val/var 变量名 = """字符串1 字符串2"""

val sql = """
        |   select
        |       *
        |   from
        |       t_user
        |   where
        |       name = "zhangsan"
"""

块表达式

val a = {
    | println("1+1")
    | 1 + 1
    | }
-------------------------
1+1
a: Int = 2

块表达式的值等于最后一个表达式的值，块中的其他代码也会被执行。

守卫

for 表达式中，可以添加 if 判断语句，这个 if 判断语句就称之为守卫。

// 打印能够整除3的数字
for(i <- 1 to 10 if i % 3 == 0) println(i)
---------------------------------------------------
3
6
9

for推导式

可以使用 for 推导式生成一个新的集合（yield）。

//生成一个10、20、30......100的集合
val v = for(i <- 1 to 10) yield i * 10

break和continue

在scala中，不能直接使用break/continue，需要scala.util.control包的Break类的breakable和break方法。

break

//打印1-100的数字，如果数字达到50，退出for表达式
import scala.util.control.Breaks._

breakable{
    for (i <- 1 to 100) {
        if (i >= 50) break()
        else println(i)
    }
}

continue

//打印1-100的数字，如果能整除10，则不打印
import scala.util.control.Breaks._

for (i <- 1 to 100) {
    breakable{
        if (i % 10 == 0) break()
        else println(i)
    }
}

方法定义

def methodName (参数名：参数类型，参数名：参数类型) : [return type] = {
    // 方法体
}
  • 参数列表的参数类型不可以省略
  • 返回值类型可以省略
  • 返回值可以不写 return, 默认为{}块表达式的值
def add(a:Int, b:Int): Int = {
    | a + b
    | }
add(1,2)
-----------------------------
Int = 3

递归方法

注意：定义递归方法不能省略返回值类型

def m1(num:Int): Int = {}
    | if (num <= 1) 1 
    | else m1(num - 1) * num
    | }

方法参数

默认参数

def add(x:Int = 0, y:Int = 0) = x + y
add()
---------------------------
Int = 0

带名参数

def add(x:Int = 0, y:Int = 0) = x + y
add(x=1)
---------------------------
Int = 1

变长参数

在参数类型后面加一个 * 号，表示参数可以是0个或者多个。

def add(num:Int*) = num.sum
add(1,2,3,4,5)
---------------------------
Int = 15
